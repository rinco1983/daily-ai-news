# 每日 AI 速递 - 项目总览

## 📋 项目简介

每日 AI 速递是一个自动化的 AI 资讯聚合平台，从主流科技媒体抓取 AI 相关新闻，提供双语阅读体验。

### 核心功能
- 📰 **新闻聚合**：TechCrunch、The Verge、VentureBeat 等
- 🌐 **双语支持**：英中双语显示
- 🎨 **ReadHub 风格**：简洁商业设计
- 🌙 **深色模式**：支持明暗主题切换
- 📱 **响应式**：完美适配各种设备

## 🏗️ 技术架构

### 后端技术
- **Python 3.9+**：主要编程语言
- **Jinja2**：模板引擎
- **Requests**：HTTP 请求
- **BeautifulSoup4**：HTML 解析
- **Vercel**：部署平台

### 前端技术
- **原生 HTML/CSS**：无框架依赖
- **CSS 变量**：主题切换支持
- **原生 JavaScript**：交互功能
- **响应式设计**：移动优先

### 数据处理
- **RSS 抓取**：自动获取新闻
- **智能分类**：大模型、AI 绘画等
- **热度计算**：AI 智能评分
- **双语翻译**：专业翻译服务

## 📁 项目结构

```
daily-ai-news/
├── api/                    # Vercel API 端点
```
