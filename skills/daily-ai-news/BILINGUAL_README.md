# 每日 AI 速递 - 双语版本 (Bilingual Version)

## 新功能：双语显示

本版本新增了英中双语显示功能，为每条英文新闻提供中文翻译，方便不同语言背景的用户阅读。

## 功能特性

### 1. 双语标题
- 英文原标题：`GPT-5 Leaks: OpenAI's Next Model to Feature Real-Time Multimodal Understanding`
- 中文翻译：`GPT-5 曝光：OpenAI 下一代模型将具备实时多模态理解能力`

### 2. 双语摘要
- 英文摘要：原文内容
- 中文摘要：专业翻译，保持准确性

### 3. 交互式切换
- 点击"显示中文"按钮查看中文翻译
- 点击"显示英文"按钮返回英文原文
- 保持阅读流畅性

### 4. 智能翻译
- 专业术语准确翻译
- 保持原文语义和上下文
- 符合中文阅读习惯

## 技术实现

### 翻译引擎
使用自定义翻译器，包括：
- 关键词映射词典
- 智能模式识别
- 预设翻译模板
- 可扩展的翻译规则

### 数据结构
```python
{
    "title": "英文标题",
    "title_cn": "中文标题",
    "text": "英文摘要",
    "text_cn": "中文摘要"
}
```

### 前端交互
- CSS 控制显示/隐藏
- JavaScript 切换语言
- 保持页面布局稳定

## 使用示例

### 页面显示效果
```
🔍 每日 AI 速递 - 2026年02月11日

┌─ 文章1 ──────────────────────────────┐
│ GPT-5 Leaks: OpenAI's Next Model...  │  ← 英文标题
│                                      │
│ GPT-5 曝光：OpenAI 下一代模型...      │  ← 中文标题
│                                      │
│ 英文摘要...                          │  ← 默认显示英文
│                                      │
│ [显示中文] 🌐                       │  ← 点击切换按钮
└─────────────────────────────────────┘
```

### 交互流程
1. 默认显示英文内容
2. 点击"显示中文"按钮
3. 切换到中文翻译
4. 点击"显示英文"按钮返回

## 添加新翻译

### 翻译器配置
在 `translator.py` 中添加新的翻译映射：

```python
title_translations = {
    "New AI News Title": "中文新闻标题",
    # ... 更多翻译
}
```

### 摘要翻译规则
```python
if "ai" in title.lower():
    summary_cn = "AI 相关的中文翻译"
elif "tech" in title.lower():
    summary_cn = "技术相关的中文翻译"
```
